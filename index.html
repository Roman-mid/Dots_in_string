Точки в строчке

Инпут принимает строку, а на выход отдает массив строк со всеми возможными размещениями точек между букв

вариантов: 0

Описание решение задачи

Оснойной принцип работы заключается в следующем. Берем строку '1234' ПЕРЕМЕЩАЕМ точку после каждого символа строки и пушим в массив. Получаем массив [‘1234’ , ‘1.234’ , ’12.34’ , ‘123.4’]. Затем берем следующий эл-т массива (1.234) и проделываем тоже самое. Получается одна точка уже стоит в строке и одну точку мы ПЕРЕДВИГАЕМ между символами. Проверяем нет ли такого элемента в массиве и нет ли 2 точек подрят (1..234) и пушим в массив. Затем берем следующий эл-т массива и повторяем. Рекурсия остановится когда в масиве появится эл-т 1.2.3.4

Реализация логики

const str = '123456' let arr = []; arr.push(str); let i = 0; function getArr(str) { const length = arr[0].length + (arr[0].length - 1); const checkLehgth = arr.some(string => string.length == length); if (checkLehgth) { return arr; } else { let j = 1; while (j < arr[i].length) { const newStr = str.slice(0, j) + '.' + str.slice(j); const checkStr = /\.\./.test(newStr); if (!arr.includes(newStr) && checkStr == false) { arr.push(newStr); }; j++; }; i++; if (arr[i]) { getArr(arr[i]); }; }; return arr; };
